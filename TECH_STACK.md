# 기술 스택

## 주요 기술 스택 (간결 버전)

NestJS, Next.js, React, TypeScript, Prisma, SQLite, Cloudinary, Tailwind CSS, Framer Motion

---

## 상세 기술 스택

### 백엔드
- **NestJS** - Node.js 백엔드 프레임워크
- **TypeScript** - 타입 안정성
- **Prisma** - ORM (데이터베이스 관리)
- **SQLite** - 데이터베이스
- **Cloudinary** - 이미지 저장 및 최적화
- **Express** - 웹 서버 (NestJS 내부 사용)
- **class-validator** - 데이터 검증
- **exifreader** - EXIF 데이터 추출

### 프론트엔드
- **Next.js** - React 프레임워크
- **React** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 CSS 프레임워크
- **Framer Motion** - 애니메이션
- **next-themes** - 다크모드 지원
- **react-masonry-css** - 갤러리 레이아웃
- **react-infinite-scroll-component** - 무한 스크롤
- **swiper** - 슬라이더/캐러셀
- **react-easy-crop** - 이미지 크롭
- **react-zoom-pan-pinch** - 이미지 확대/축소
- **lottie-react** - Lottie 애니메이션
- **canvas-confetti** - 파티클 효과
- **react-dropzone** - 파일 드래그 앤 드롭
- **lucide-react** - 아이콘
- **pretendard** - 한글 폰트
