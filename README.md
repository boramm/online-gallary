# 시선이 잇(있)는 날 📸

직원 문화행사 - 사진 공모전 온라인 갤러리

## 🎯 프로젝트 소개
- 대학 직원들의 일상과 시선을 담은 사진 공모전 플랫폼
- "connection(잇다)"과 "being(있다)"의 언어유희를 담은 프로젝트

## 🛠 기술 스택
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: NestJS, Node.js
- **Database**: PostgreSQL / SQLite
- **Image**: Cloudinary

## ✨ 주요 기능
- 부서별 필터링
- 실시간 좋아요 & 조회수 (IP 기반 중복 방지)
- 댓글 시스템
- 관리자 모드 (사진 수정/삭제)
- 반응형 디자인

## 🚀 실행 방법
```bash
npm install
npm run dev
```
